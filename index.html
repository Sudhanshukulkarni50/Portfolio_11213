CloudPulse Intelligence Platform

Serverless AWS platform that monitors EC2 utilization using CloudWatch metrics and predicts future infrastructure needs.

  • Analyzes last 7 days of CPU metrics
  • Detects underutilized & overloaded EC2 instances
  • Predicts utilization patterns
  • Generates automated infrastructure recommendations
Impact & Results
  • Monitors multiple EC2 instances with 7-day rolling metrics
  • Automated alerts reduce manual infrastructure review time
  • SNS notifications trigger within seconds of threshold breach
  • Serverless architecture runs at near-zero idle cost
How It Works
  1. EventBridge triggers Lambda on a scheduled interval
  2. Lambda queries CloudWatch for EC2 CPU metrics (last 7 days)
  3. Metrics are analyzed for under/over-utilization thresholds
  4. Recommendations and anomaly flags are stored in DynamoDB
  5. SNS sends automated alerts; API Gateway exposes results
AWS Lambda API Gateway CloudWatch EventBridge SNS DynamoDB

Aadhaar Enrolment & Update Analytics System

Large-scale data analytics pipeline analyzing 18+ lakh anonymized Aadhaar records across multiple states of India.

  • Data cleaning and transformation pipeline
  • Multi-state enrollment analytics
  • Interactive Power BI dashboards
Impact & Results
  • 18+ lakh (1.8M+) anonymized records processed end-to-end
  • Multi-state enrollment trends visualized across India
  • Data quality pipeline eliminated manual correction effort
  • Power BI dashboards enable real-time enrollment insights
How It Works
  1. Raw Aadhaar enrollment CSV data is ingested into AWS S3
  2. Python scripts clean, normalize, and deduplicate records
  3. Transformed data is loaded into PostgreSQL for querying
  4. SQL views and aggregations prepare data for reporting
  5. Power BI connects to PostgreSQL and renders dashboards

India JobStream

Automated data pipeline that ingests thousands of job postings daily and generates insights about hiring trends.

  • 5000+ job postings ingested daily
  • Snowpipe automated ingestion
  • Hiring trends and salary analytics dashboards
Impact & Results
  • 5,000+ job records ingested and processed daily via automation
  • Snowpipe enables near-real-time data availability in Snowflake
  • Salary and demand trends surfaced across 10+ job categories
  • Fully automated pipeline with zero manual intervention
How It Works
  1. Python scraper collects job listings from multiple platforms
  2. Raw JSON/CSV files are staged in Azure Blob Storage
  3. Snowpipe auto-ingests new files into Snowflake staging tables
  4. dbt/SQL transforms data into analytics-ready fact tables
  5. Power BI reads from Snowflake and renders hiring trend reports
